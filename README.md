## 🔑 Основні особливості проекту
 * Мультивалютність: Підтримка рахунків у різних валютах (UAH, USD, EUR) з автоматичним відображенням знаків валют.
 * Гібридна модель створення рахунків:
1. Ручний облік: можливість створювати готівкові "гаманці" або рахунки, де баланс оновлюється виключно користувачем.
2. API-синхронізація: автоматичне підключення банківських карток через API Monobank з імпортом актуального стану балансу.
* Управління операціями: Створення записів про доходи та витрати з можливістю деталізації (дата, сума, опис, категорія).
 * Розумна категоризація: Система автоматично запам'ятовує введені користувачем категорії і пропонує їх у вигляді списку автозаповнення, що мінімізує помилки при введенні.
 * Excel-style фільтрація: Функціонал розширеного пошуку та фільтрації історії операцій за типом (дохід/витрата) та ключовими словами (опис/категорія), що дозволяє швидко знаходити конкретні платежі.
 * Моніторинг валютного ринку: Автоматизована інтеграція з API НБУ для отримання офіційних курсів валют, що дозволяє користувачу розуміти актуальну вартість своїх валютних активів.
 * Автономна синхронізація: Механізм періодичного оновлення банківських даних, що забезпечує актуальність інформації без необхідності ручного звіряння з додатком банку.

## 📂 Структура проекту
 * `web/` — клієнтська частина (frontend), розроблена з використанням бібліотеки React та інструменту збирання застосунків Vite. Забезпечує інтерактивний інтерфейс, маршрутизацію та відображення фінансової аналітики.
 * `server/` — серверна частина (backend), створена на Node.js з використанням фреймворку Express.js. Виконує обробку запитів, бізнес-логіку синхронізації та взаємодію з зовнішніми API.



## 🚀 Як розгорнути і запустити проект
### Вимоги
- Node.js (версія 18+ для підтримки вбудованого fetch)
- NPM (Node Package Manager)
- MongoDB (доступ до MongoDB Atlas або локальне використання)

### Інструкції
1) **Клонування проекту**
-
  Клонувати репозиторій на локальну машину
   ```
   git clone https://github.com/vtkachenko157/finance-app.git
   ```
   Перехід в папку проекту
   ```
   cd finance-app
   ```
2) **Запуск бекенду (серверної частини)**
-
   Перейти у папку серверу (server) та встановити залежності командою:
   ```
   npm install
   ```
-
   Запуск серверу:
   ```
   npm run start
   ```
3) **Запуск фронтенду (клієнтської частини)**
-
   Перейти у папку web та встановити залежності командою:
   ```
   npm install
   ```
-
   Запуск React додатку:
   ```
   npm run dev
   ```

Після запуску застосунок буде доступний за посиланням в браузері - `http://localhost:5173`
